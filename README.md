# WordPress Plugins

This repository contains WordPress plugins developed by Vicedomini Softworks.

- WordPress 7.0+, PHP 8.1+
- GPL v3

## Plugins

| Plugin | Description | Docs |
|--------|-------------|------|
| **VSecrets Manager** | Manage secrets across encrypted database, AWS Secrets Manager, and HashiCorp Vault/OpenBao providers | [v-secrets-manager/README.md](v-secrets-manager/README.md) |
| **Social Feed** | Display social media feeds from Instagram, Facebook, TikTok, X, Threads, Bluesky, and YouTube (requires v-secrets-manager) | [social-feed/README.md](social-feed/README.md) |
| **VS Mailer** | SMTP, Brevo, and Mailgun email delivery (requires v-secrets-manager) | [vs-mailer/README.md](vs-mailer/README.md) |

Each plugin's README has its own setup, feature list, linting, testing, and CI/CD details.
